
# Simulador de Satèl·lit

Aquest és un projecte inicial per simular les dades bàsiques d'un satèl·lit mitjançant Python. 

## Funcionalitats
- Simulació de telemetria en temps real.
- Monitorització de paràmetres crítics:
  - **Temperatura:** Control de la temperatura interna (en ºC).
  - **Bateria:** Estat de càrrega del sistema (en %).

## Com executar-lo
1. Clona el repositori:
   `git clone https://github.com/sanjothecoder/simulador-satelit.git`
2. Entra a la carpeta:
   `cd simulador-satelit`
3. Executa el script:
   `python3 sat_sim.py`

## Objectiu
Aquest projecte serveix com a base per aprendre sobre gestió de dades i telemetria en sistemes espacials.
